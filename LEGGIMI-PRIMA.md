# LEGGIMI PRIMA 🌿

Questa è la versione **commentata + restyling immersivo + light/dark mode** del progetto Albergo.

Per studiarla senza perderti:

1. apri `GUIDA-CODICE-COMMENTATO.md`;
2. guarda `frontend/src/app/homepage/` per hero immersiva, Material e animazioni accessibili;
3. apri `GUIDA-TEMA-LIGHT-DARK.md` e confrontalo con `core/services/theme.service.ts`;
4. passa a `rooms/rooms.component.ts` per vedere signal, computed, form e service insieme;
5. segui una chiamata dentro `core/services`;
6. trova l'endpoint corrispondente in `backend/.../Controller`;
7. guarda la relativa entity in `backend/.../Model`.

I commenti didattici aggiunti sono riconoscibili dalle etichette:

```text
GUIDA DIDATTICA DEL FILE
GUIDA DIDATTICA TEMPLATE
GUIDA DIDATTICA STILI
GUIDA DIDATTICA BACKEND
BLOCCO DIDATTICO
```

Per l'avvio usa `README-AVVIO.md`.


Per una mappa veloce dei componenti usa `MAPPA-COMPONENTI-E-BLOCCHI.md`.
Per il riepilogo di questa versione usa `AGGIORNAMENTI-22-07-TEMA.md`.
